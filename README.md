# Vite React TypeScript Project

This project is a Vite-based React application written in TypeScript.

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Lint the project:

```bash
npm run lint
```

Type-check the project:

```bash
npm run typecheck
```

## Project structure

- `src/` — application source code
- `src/components/scenes/` — React scene components
- `src/hooks/` — custom hooks
- `src/utils/` — utility modules
- `vite.config.ts` — Vite configuration
- `tailwind.config.js` — Tailwind CSS configuration

## GitHub

The project is pushed to the configured GitHub remote and tracking branch `main`.
